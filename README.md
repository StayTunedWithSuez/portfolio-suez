#  React Portfolio Site

This is my personal portfolio website built using **React** and modern web technologies to showcase my skills, projects, and experience.

##  Features

- Responsive design (Mobile + Desktop friendly)
- Modern UI with clean layout
- Project showcase section
- About & Skills section
- Contact section
- Fast performance with React components

## Tech Stack

- React
- Tailwind CSS 
- React Router 
- Vite 

## Installation

```bash
git clone https://github.com/StayTunedWithSuez/portfolio-suez.git
cd <project-folder>
npm install
npm run dev
